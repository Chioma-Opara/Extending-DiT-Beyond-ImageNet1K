# Extending-DiT-Beyond-ImageNet1K
Fine-Tuning a Diffusion Transformer on a Custom ImageNet Subset Under Limited Compute Constraints

Abstract
Diffusion models have recently become one of the dominant approaches for high-quality image generation, with Diffusion Transformers (DiTs) demonstrating strong performance by replacing traditional U-Net backbones with transformer architectures. This project explores whether a pretrained DiT model can be successfully fine-tuned on image categories outside those it was originally trained on. I focus not only on model training, but also on understanding the underlying architecture, constructing a usable dataset, modifying an existing training pipeline, and evaluating the behavior of the model under significant computational and storage constraints.

This repository contains relevant scripts and Jupyter notebooks used for this project
